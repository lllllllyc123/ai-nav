## ➕ 如何新增工具

编辑 `data.js`，在对应分类数组末尾加对象：

```js
{
    name: "工具名称",
    desc: "一句话描述",
    url: "https://example.com",
    tags: ["标签1", "标签2"],
    category: "writing",     // 见下表
    pricing: "free",         // free | freemium | paid
    isHot: false,            // 热门推荐
    isNew: true,             // 本周新增
}
```

**分类 key**：

| key | 名称 | 工具数 |
|:---|:---|:---|
| `writing` | AI写作 | 8 |
| `image` | AI绘画/图像 | 7 |
| `video` | AI视频 | 6 |
| `office` | AI办公/效率 | 6 |
| `coding` | AI编程 | 4 |
| `marketing` | AI营销/电商 | 4 |
| `academic` | AI学术 | 5 |
| `other` | 其他AI工具 | 4 |

> 💡 每周五固定更新 3-5 个新工具，设 `isNew: true`，并在社媒同步"本周新增"通知。

## 📢 运营路线图

### 现在可做的（变现优先，不需要登录）

按 ch0023 的核心结论，导航站赚钱靠三件事，都不需要用户登录：

```
流量 → CPS 佣金（用户点链接跳转，拿提成 10%-30%）
流量 → 广告坑位费（日 IP 300+ 后，首页位 ¥300-800/月）
流量 → 私域沉淀（首页加微信二维码 → 群里深度变现）
```

### 第一阶段：冷启动（前 1-2 周）

1. **私域邀约**：链接发给微信好友，免费试用，收集反馈
2. **论坛截流**：去 V2EX、知乎校园板块发"整理了 30 个 AI 工具"干货帖，软植入链接
3. **社群分享**：技术群/学生群/电商群，有人问工具时丢出导航站页面

### 第二阶段：低成本获客

- **SEO 长尾词**：布局"免费 AI PPT 生成器推荐""2025 程序员 AI 工具导航"
- **小红书/抖音**：工具横评内容，文末留导航站链接
- **每周更新通知**：公众号 / 朋友圈发"本周新增 5 个工具"

### 第三阶段：变现

| 方式 | 如何做 | 预期 |
|:---|:---|:---|
| CPS 佣金 | data.js 的 url 换成返利链接 | ¥1,000-5,000/月 |
| 固定坑位费 | 首页推荐位按月出租 | ¥300-800/位/月 |
| 私域变现 | 加微信 → 群 → 卖课/社群 | ¥3,000+/月 |

### CPS 实操

1. 去工具官网找 "Affiliate" / "合作伙伴" 页面
2. 注册获取推广链接
3. 替换 `data.js` 中的 `url`：

```js
// 改前
{ name: "Jasper", url: "https://www.jasper.ai" }

// 改后（Jasper 返利 30%）
{ name: "Jasper", url: "https://www.jasper.ai?ref=你的推广码" }
```

> ⚠️ 国内平台发布时部分海外链接可能被屏蔽，优先用国内可访问的镜像。

## ❌ 不做的事情（有明确理由）

| 事项 | 理由 |
|:---|:---|
| 用户登录/注册 | 导航站变现不需要登录，加登录只会增加跳出率和维护成本 |
| 手机号验证 | 国内短信 ~¥0.045/条，无收入前纯烧钱 |
| 后端数据库 | 纯静态站够用，工具数据量不大用 JSON 即可 |
| 自建服务器 | tiiny.host / GitHub Pages 免费够用到日 UV 破万 |

> 以上决策来自 brainstorming 过程，遵循 ch0023「最稳、最简单、最少改动」原则。
